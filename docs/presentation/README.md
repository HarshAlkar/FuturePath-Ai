# FuturePath AI - Presentation

## Generate PPTX

### Prerequisites
- Node.js 18+
- Internet access to install dependencies

### Steps
1. Open a terminal in the project root.
2. Install dependency:
   ```bash
   npm install pptxgenjs
   ```
3. Run the generator script:
   ```bash
   node docs/presentation/generate_ppt.js
   ```
4. Output will be created at:
   ```
   FuturePathAI_Presentation.pptx
   ```

### Slides Included
- Title
- Problem Statement
- Solution / Idea
- Literature Survey
- Key Features
- Architecture (High-Level)
- AI Insights
- OCR Receipt Detection
- Stock Analyzer
- APIs (Summary)
- Testing & Troubleshooting
- Performance & Security
- Roadmap
- Team
- Thank You / Q&A

To customize content, edit `docs/presentation/generate_ppt.js`.
